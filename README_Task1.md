# 🌐 Language Translation Tool — CodeAlpha AI Internship Task 1

A Python desktop application that translates text between 100+ languages using Google Translate API.

## 📸 Features
- Translate between 100+ languages
- Auto language detection support
- ⇄ Swap source and target languages instantly
- 📋 One-click copy translated result
- Character counter (0 / 5000)
- Clean dark-themed UI built with Tkinter

## 🛠️ Tech Stack
- **Python 3.x**
- **Tkinter** — GUI
- **googletrans 4.0.0-rc1** — Google Translate API wrapper

## 🚀 How to Run

### 1. Install dependencies
```bash
pip install googletrans==4.0.0-rc1
```

### 2. Run the app
```bash
python translator.py
```

## 📁 Project Structure
```
CodeAlpha_LanguageTranslator/
│
└── translator.py       # Main application file
```

## 📌 Usage
1. Type or paste text in the left box
2. Select source and target languages from dropdowns
3. Click **Translate** or press **Enter**
4. Translated text appears on the right
5. Use **Copy** to copy the result
